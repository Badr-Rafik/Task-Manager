# Task Manager

A simple command-line task management application that lets you organize and track your daily tasks with persistent JSON storage.

## Features

- **Add Tasks** - Create new tasks with automatic ID generation
- **View Tasks** - Display all tasks with completion status at a glance
- **Update Tasks** - Modify task titles by task ID
- **Delete Tasks** - Remove tasks from your list
- **Mark as Completed** - Track task progress by marking tasks as done
- **Persistent Storage** - All tasks are automatically saved to `tasks.json`

## Requirements

- Python 3.x
